# Quest Builder
Randomly generates quests and is written completely lua!

Example:
```
--------
QUEST ID: 1
Collect 2 ring(s).
REWARD: A shovel
--------
QUEST ID: 2
Rescue a frog from a dragon!
REWARD: A sword
--------
QUEST ID: 3
Collect 1 gold coin(s).
REWARD: A sword
--------
QUEST ID: 4
Collect 10 sword(s).
REWARD: A gold coin
--------
a or add - add in a new quest.
r or remove - remove quest by id.
q or quit - exit program.

```
